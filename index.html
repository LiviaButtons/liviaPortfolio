Weekly's

About this project

This is a school project completed over a period of two months in a group of four. The brief? Create a website from scratch, based on an original idea, brand, and concept. We decided to build a recipe website, but with an additional feature that would set us apart from the competition online: a generator for a weekly calendar to build a menu for the week. Our target audience? Young people and families who want to take the stress out of cooking and having to plan meals.

Once the concept was decided, we prepared a project brief and built the brand (logo, name, tagline, colours), as well as a prototype in Adobe XD to test the look and user experience of the future website. Then, we built the database and responsive website in about 2-3 weeks.

Features I built include:

  • search function with autocomplete
  • calendar generator
  • mobile calendar
  • contact form with validation in JS and PHP
  • menu responding to scroll
  • mobile hamburger menu

RoboCode

About this project

During a three-day Hackathon organised by Interface3, students in back-end, front-end and game development came together in groups to create projects under the theme of "Decode & Code". The goal? Create a game, website or app that inspires people to want to learn how to code.

Our team of six (one back-end, two game and three front-end developers) decided to build RoboCode, a fun browser-game targetted at 5-10 year-olds. Over five levels, they need to complete commands (in mock-Javascript) to make the character "Sparky" move. Built in just three days, we delivered a fully functional prototype at the end of the Hackathon.

Features I built include:

  • drag and drop function
  • animation of progress bar
  • validation of answers
  • level design

Logo for Weekly Chef

About this project

This is a logo and name proposal that I created for the brand of a recipe website built for a school project. The website's main feature is a calendar generator that prepares a weekly meal plan for the user. The name "Weekly Chef" is meant to highlight that feature, and suggest to users that they, too, can cook like a chef for a week with the help of our website. The tagline "plan it, cook it, eat it" serves as a quick explainer for what the website does. The chosen fonts were Century Gothic and Lato Thin.

The Octopus in the logo takes on the role of the chef. Having eight arms to juggle all the tasks needed while cooking is an advantage, of course - but with the help of our website, our users will manage just as well. The calendar that he's holding, of course, leads back to the themes of planning and preparedness. The coral, green and orange are colours from the website palette that I integrated into this logo. I also prepared differently coloured alternatives to the Octopus, as well as black and white alternatives.

The octopus could be used as a mascot as well, to appear in promotional videos, tutorials, and chats. I also prepared a set of icons to go along with the octopus and that could be used throughout the website, creating a cohesive overall brand and image.

Tools

  • pen and paper
  • Adobe Illustrator

Pixabay Photography

About this project

After a month-long "Start & Code" module with first classes in HTML, CSS, JavaScript and PhotoShop, this is the first website I ever built. The purpose of this project was to showcase the skills acquired in that month. Unfortunately, these skills didn't yet include responsive web design, so despite some attempts with the help of online resources, the website is desktop-only.

I decided to build the website for a fictional photographer who specialises in portraits of humans and their pets. This would give me the opportunity to experiment with various different tools to showcase those images (gallery, lightbox), as well as build several different kinds of pages while maintaining a coherent look. The pictures were sourced from Pixabay, inspiring the name.

Features I included:

  • Parallax scroll built in pure CSS
  • lightbox galleries
  • form responding to user input
  • Logo design
